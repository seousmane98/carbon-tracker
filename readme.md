# Carbon Tracker Dashboard

Une API Python pour calculer et visualiser l'empreinte carbone.

## Description

Carbon Tracker est un tableau de bord qui aide à :
- Calculer l'empreinte carbone d'une activité
- Visualiser les données en temps réel
- Comprendre l'impact environnemental

## Technologies utilisées

- **Backend** : Python (HTTP Server)
- **Frontend** : React (à venir)
- **Cloud** : AWS
- **Version Control** : Git + GitHub

## Installation

```bash
git clone <ton-repo>
cd carbon-tracker
python main.py
```

Puis ouvre : `http://localhost:8000`
